# CrownMetrics

A data driven look at the careers and legacy of Jordan and Lebron

I chose this project because the debate about who is the greatest NBA player of all time has been raging for years, but I was never satisfied with the methods of comparison. 

Using www.basketball-reference.com I was able to download all NBA League Data as well as individual player data and convert that data to an excel file. I chose to limit the data to 11 specific seasons. I chose 11 seasons because that was the number of full uninterrupted seasons played by the player (Michael Jordan) who played fewer seasons. I did this because the goal was to compare player dominance based on performance and not based on longevity. This prevented stats from being skewed after Jordan retired. I also took care to choose seasons for each player where the player played all games. This means there is not a data set where a player is out for a significant number of games. 

I combined each player’s individual season stats over the 11 chosen seasons and combined the league data for all of those seasons to get a percentage of each player's stats compared to the rest of the league in 5 major categories: Points, Steals, Assists, Blocks, and rebounds. I then used the league data to calculate changes in scoring and 3 point attempts over time. 

Once the data was cleaned and organized, I used Power BI to create visuals for the main data points.
